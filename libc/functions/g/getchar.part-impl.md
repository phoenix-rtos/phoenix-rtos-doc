# Synopsis

`#include <stdio.h>`

`int getchar(void);`

## Status

Partially implemented

## Conformance

IEEE Std 1003.1-2017

## Description

The purpose is to get a byte from a stdin stream. The `getchar()` function shall be equivalent to `getc(stdin)`.

## Return value

Refer to [fgetc](../f/fgetc.part-impl.md).
