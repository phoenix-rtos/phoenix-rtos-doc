# dmesg

`dmesg` read messages from the kernel's ring buffer.

## Usage

```console
dmesg [options]
```

### Options

- `-D`: Disables the printing of messages to the console.
- `-E`: Enables the printing of messages to the console.
- `-h`: Displays help information,
